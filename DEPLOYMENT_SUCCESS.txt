  🚀 UTILISATION IMMÉDIATE
================================================================================

1️⃣ LANCER LE SERVEUR AI (avec protection intégrée)

   powershell -ExecutionPolicy Bypass -File .\start_ai_server.ps1

   Ou directement :
   python backend/api/main.py

   La bannière affichera :
   ┌───────────────────────────────────────────────────────────────┐
   │ 🚀 TRADBOT AI SERVER - DÉMARRAGE                              │
   │ 🛡️  SYSTÈME DE PROTECTION DES PERTES ACTIVÉ                   │
   │    • Perte maximale par trade: 3.00 USD                       │
   │    • Monitoring API disponible                                │
   └───────────────────────────────────────────────────────────────┘


2️⃣ LANCER LE MONITORING CONTINU (recommandé)

   Dans un terminal séparé :
   powershell -ExecutionPolicy Bypass -File .\start_loss_monitor.ps1

   Le monitoring surveillera automatiquement TOUTES les positions
   et fermera celles qui dépassent -3.00 USD


3️⃣ TESTER L'API

   # Vérifier le statut
   curl http://localhost:8000/monitor/status

   # Déclencher un monitoring manuel
   curl -X POST http://localhost:8000/monitor/loss-protection?max_loss=3.0

   # Via le endpoint robot (identique)
   curl -X POST http://localhost:8000/robot/monitor/loss-limit?max_loss=3.0
