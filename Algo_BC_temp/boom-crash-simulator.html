IsAlgoSpikeReady() — 3 conditions

// 1. Z-Score ATR (fenetre 20)
mean = average(ATR[1..20])
stdDev = sqrt(variance(ATR[1..20]))
zScore = (ATR[0] - mean) / stdDev
atrReady = zScore ≤ -1.0 OR zScore ≥ 2.0

// 2. EMA(14) directionnelle
ema14 = EMA(close, 14)
Boom:  emaOk = close > ema14
Crash: emaOk = close < ema14

// 3. Cooldown (pas de spike recent)
maxMove(5 bars) < avgBody(10 bars) × 3.0

Gates universelles (toutes entrees)

1. GOM Coherence ≥ 70%
2. Boom = BUY only / Crash = SELL only
3. Max 1 position par symbole
4. Capital Manager (profit/perte jour)
5. SetupScore ≥ 65
6. 7min no-profit auto-close
7. GOM WAIT = fermer positions

Boom 300:  ~135 ticks de pause
