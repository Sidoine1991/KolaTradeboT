ÉTAPES DE COMPILATION:
═══════════════════════════════════════════════════════════════════

1️⃣  VÉRIFIER LES FICHIERS
    ✓ SMC_Advanced_Entry_System.mqh       (d:\Dev\TradBOT\)
    ✓ SMC_Enhanced_Entry_Integration.mqh  (d:\Dev\TradBOT\)
    ✓ SMC_Universal.mq5                   (d:\Dev\TradBOT\ - MODIFIÉ)

2️⃣  OUVRIR MT5
    → File → Open → SMC_Universal.mq5

3️⃣  VÉRIFIER LES INCLUDES
    La section des includes doit contenir:
    
    #include <Trade/Trade.mqh>
    #include <Trade/PositionInfo.mqh>
    #include <Trade/OrderInfo.mqh>
    #include <Trade/DealInfo.mqh>
    #include <Trade/HistoryOrderInfo.mqh>
    
    // NOUVEAUX: Système d'entrée avancé basé sur Price Action
    #include "SMC_Advanced_Entry_System.mqh"
    #include "SMC_Enhanced_Entry_Integration.mqh"

4️⃣  COMPILER (Ctrl+Shift+F9)
    
    Résultats attendus:
    ✓ 0 errors
    ✓ Warnings possibles (ignorables pour cette version)
    
    Erreurs possibles & solutions:
    ❌ "Cannot open include" 
       → Vérifier chemins des fichiers .mqh
       → Placer les fichiers dans le même dossier que SMC_Universal.mq5
    
    ❌ "Undefined function"
       → Vérifier que les prototypes de fonctions existants sont accessibles
       → (Ex: CalculateLotSize(), ValidateAndAdjustStopLossTakeProfit, etc.)

5️⃣  RAFRAÎCHIR EXPERT ADVISORS (MT5)
    → Navigator (Ctrl+N)
    → Expert Advisors
    → Clic droit → Refresh
    → Le robot doit réapparaître

6️⃣  ATTACHER LE ROBOT AU GRAPHIQUE
    → Ouvrir un graphique (ex: Boom1M)
    → Drag & drop SMC_Universal.mq5 sur le graphique
    → Ou: Insert → Expert Advisors → SMC_Universal

7️⃣  CONFIGURER LES PARAMÈTRES
    
    Dans la fenêtre "Expert Advisors → SMC_Universal":
    
    ╔═════════════════════════════════════════════════════════════════╗
    ║ ADVANCED PRICE ACTION ENTRY SYSTEM - Paramètres               ║
    ╚═════════════════════════════════════════════════════════════════╝
    
    📌 UseAdvancedPriceActionEntry = TRUE
       ↳ Activer le nouveau système (recommandé)
    
    📌 AdvancedEntryMinimumScorePercent = 75.0
       ↳ Score minimum pour entrer
       ↳ 75.0 = bon compromis
       ↳ 80.0 = très strict (moins de trades)
       ↳ 70.0 = moins strict (plus de trades)
    
    📌 AdvancedEntryRequireMultiTimeframeConfluence = TRUE
       ↳ Exiger confluence M1+M5+H1
       ↳ TRUE = plus fiable (recommandé pour Forex)
       ↳ FALSE = plus de setups mais moins de qualité
    
    📌 AdvancedEntryUseEngulfing = TRUE
       ↳ Activer détection Engulfing
    
    📌 AdvancedEntryUsePinBar = TRUE
       ↳ Activer détection Pin Bar
    
    📌 AdvancedEntryUseInsideBar = TRUE
       ↳ Activer détection Inside Bar
    
    📌 AdvancedEntryUseHarami = TRUE
       ↳ Activer détection Harami
    
    📌 AdvancedEntryStopLossMultiplier = 1.5
       ↳ Distance SL = 1.5 × ATR
       ↳ Recommandé: 1.5 - 2.0
    
    📌 AdvancedEntryTakeProfitMultiplier = 3.0
       ↳ Distance TP = 3.0 × ATR
       ↳ Recommandé: 2.5 - 3.5
    
    📌 AdvancedEntryLogPatternDetails = TRUE
       ↳ Afficher logs détaillés
       ↳ TRUE = debug (voir patterns détectés)
       ↳ FALSE = moins de spam

8️⃣  VÉRIFIER LA CONSOLE (View → Experts/Toolbox)
    
    Logs attendus (toutes les 2-5 secondes):
    
    ✅ "SETUP ACCEPTED - BUY | Pattern: 82.0% | Confluence: 78.0%"
       → Setup valide, entrée probable
    
    ⚠️  "SETUP REJECTED - SELL | Score: 62.5% < 75.0%"
       → Setup rejeté (score insuffisant)
    
    🎯 "ENTRY EXECUTED - BUY | Entry: 29450 | SL: 29380 | TP: 29590"
       → Entrée exécutée avec succès

TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════

❌ PROBLÈME: "Erreur compilation - Cannot open include"
   ✅ SOLUTION: 
      - Placer les 2 fichiers .mqh dans le dossier:
        C:\Users\[USER]\AppData\Roaming\MetaTrader 5\MQL5\Include\
      - OU dans le même dossier que SMC_Universal.mq5

❌ PROBLÈME: "Aucune entrée Advanced PA, que des anciennes entrées"
   ✅ SOLUTION:
      - Vérifier que UseAdvancedPriceActionEntry = TRUE
      - Vérifier que AdvancedEntryLogPatternDetails = TRUE
      - Regarder les logs: chercher "SETUP REJECTED" ou "SETUP ACCEPTED"
      - Réduire temporairement AdvancedEntryMinimumScorePercent à 65%
      - Activer tous les patterns (Engulfing, PinBar, InsideBar, Harami)

❌ PROBLÈME: "Trop d'entrées, beaucoup de pertes"
   ✅ SOLUTION:
      - Augmenter AdvancedEntryMinimumScorePercent à 80%
      - Activer AdvancedEntryRequireMultiTimeframeConfluence = TRUE
      - Désactiver les patterns moins fiables (teste Harami/InsideBar)
      - Augmenter StopLossMultiplier à 2.0 ou 2.5
      - Voir les logs: chercher les patterns avec score < 70%

❌ PROBLÈME: "Compilation réussie mais le robot ne s'attache pas"
   ✅ SOLUTION:
      - Vérifier que le robot a la permission d'accéder aux données du compte
      - Redémarrer MT5
      - Vérifier l'onglet "Experts" pour les erreurs
      - Vérifier que tous les symboles définis existent
