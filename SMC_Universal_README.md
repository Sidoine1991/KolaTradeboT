# SMC Universal - Robot Smart Money Concepts (unique)

Robot de trading basé sur les concepts SMC/ICT, compatible multi-actifs avec intégration IA. **Un seul robot** pour tous les actifs.

## Concepts implémentés

FVG | OB | BOS | LS | OTE | EQH/EQL | P/D | LO/NYO | SSL/BSL | SL/TP/BE

## Actifs

Boom/Crash | Volatility | Forex | Commodities | Metals (Or, Argent)

## Installation

1. `SMC_Universal.mq5` → `MQL5/Experts/`
2. `Include/SMC_Concepts.mqh` → `MQL5/Include/`
3. WebRequest : ajouter `http://localhost:8000` et `https://kolatradebot.onrender.com`
4. Compiler (F7)
