INSTRUCTIONS: Mettre à jour le mot de passe dans .env

1. Ouvrez le fichier: D:\Dev\TradBOT\.env

2. Cherchez ces lignes (ajoutées à la fin):
   AWS_RDS_PASSWORD=YOUR_PASSWORD_HERE
   DATABASE_URL=postgresql://dbadmin:YOUR_PASSWORD_HERE@trading-db...

3. Remplacez YOUR_PASSWORD_HERE par votre vrai mot de passe

4. Sauvegardez le fichier

5. Ensuite, exécutez:
   cd D:\Dev\TradBOT
   python migrate_to_aws_rds.py

IMPORTANT: Le mot de passe sera demandé interactivement par le script de migration.
