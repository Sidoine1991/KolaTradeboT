================================================================================
TROUBLESHOOTING
================================================================================

Problem: Dashboard shows "DISCONNECTED"
Solution:
   1. Verify AI server is running: python ai_server.py
   2. Check http://127.0.0.1:8000/health in browser
   3. If localhost fails, Render will be used automatically
   4. Verify internet connection

Problem: Confidence showing 0%
Solution:
   1. Dashboard updates every 2 seconds, wait a moment
   2. Check if signal is actually being detected (look for divergence zone)
   3. Check Expert logs (Alt+L) for error messages
   4. Verify AI server is responding to requests

Problem: No divergence zones appearing
Solution:
   1. Check if RSI shows divergence patterns
   2. Compare price and RSI peaks visually
   3. Verify DivergenceStrength input (default: 20 bars)
   4. Check logs for detection attempts

Problem: Trades not executing
Solution:
   1. Verify EnableAutoTrading = true
   2. Check daily loss limit hasn't been reached
   3. Verify market is open (not Asian session 22:00-08:00)
   4. Check if all gates are passing (look for [GATE] messages in logs)
   5. Verify account has enough margin for position
