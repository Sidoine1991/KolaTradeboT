WHAT GETS CREATED
================================================================================

After each run, 3 files are created:

1. gom_signal.json (for TradeManager)
   • verdict: "SELL" / "BUY" / "WAIT"
   • score_buy, score_sell: Confluence scores
   • decision: Formatted emoji + text

2. opportunities.json (for TradeManager)
   • 3 opportunities: OPP-001, OPP-002, OPP-003
   • Each has: type, entry, sl, tp, confidence
   • TradeManager reads and executes trades

3. whatsapp_alerts.log (fallback)
   • If PsychoBot offline, messages saved here
   • Timestamped entries
