Capital de trading : 20 USD (compte Deriv, indices synthetiques Boom/Crash principalement).

**Why:** Capital tres petit = chaque centime compte. Une serie de 3 pertes a 2.5%/trade = -15% capital = difficulte a recuperer. Strategie = 1 bon trade/jour bien capture suffit.

**How to apply:**
- Lot size: toujours 0.01 (minimum broker)
- Risque/trade: 0.20$ max (1% du capital)
- Drawdown journalier max: 1.50$ (7.5%)
- Perte totale max: 3$ (15%) avant arret complet
- TP realiste: 0.50$ par trade (2.5%) avec lot 0.01 sur spike Boom/Crash
- MaxPositions terminal: 1 seule
- MaxDailyTrades: 3 (qualite > quantite)
- Confiance IA minimum: 72-82% selon le module
- Trailing stop active des 0.05$ de profit
- Salvage bank arme a 1.50$ de gain journalier
