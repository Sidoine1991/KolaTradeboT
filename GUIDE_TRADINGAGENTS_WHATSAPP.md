# Guide d'Envoi des Rapports TradingAgents via WhatsApp

## 📋 Vue d'ensemble

Le script `send_tradingagents_report.py` permet d'envoyer automatiquement les rapports TradingAgents sur WhatsApp via PsychoBot.

Deux modes disponibles:
1. **Signal rapide** : Message markdown avec niveaux de trade
2. **Rapport complet** : Fichier Word (.docx) en pièce jointe + résumé

---

## 🚀 Mode 1 : Signal Rapide

Envoie un signal formaté en markdown avec les niveaux clés.

### Commande:

```bash
python Python/send_tradingagents_report.py --quick \
  --symbol XAUUSD \
  --decision SELL \
  --entry 4570 \
  --sl 4590 \
  --tp 4550 \
  --rating "Underweight" \
  --summary "Momentum baissier confirmé par divergence baissière" \
  --rr "1:2"
```

## 📄 Mode 2 : Rapport Word Complet

Envoie le fichier Word (.docx) en pièce jointe avec un message résumé.

### Commande:

```bash
python Python/send_tradingagents_report.py \
  --file "C:/Users/USER/Documents/Rapport_XAUUSD_2026_05_25.docx" \
  --send-file
```

### Résultat WhatsApp:

1. **Pièce jointe** : Fichier `Rapport_XAUUSD_2026_05_25.docx`
2. **Message** :
   ```
   📊 *RAPPORT TRADINGAGENTS*

   Rapport_XAUUSD_2026_05_25

   Voir le résumé ci-dessous ↓
   ```
3. **Résumé** :
   ```
   📊 *RAPPORT TRADINGAGENTS*

   Fichier: *Rapport_XAUUSD_2026_05_25.docx*

   Le rapport complet a été envoyé en pièce jointe.
   Consultez le document Word pour l'analyse détaillée.
   ```

## 📝 Mode 3 : Rapport Markdown

Envoie le contenu d'un fichier `.md` ou `.txt` découpé en messages de 4000 caractères.

### Commande:

```bash
python Python/send_tradingagents_report.py \
  --file "analyses/rapport_xauusd.md"
```

### Résultat:

- Si le rapport fait moins de 4000 caractères : **1 message**
- Si plus grand : **plusieurs messages** numérotés `[1/3]`, `[2/3]`, `[3/3]`

## 🔧 Endpoints PsychoBot

Le script utilise deux endpoints:

### 1. `/send-message` (texte seulement)

```json
POST https://psychobot-1si7.onrender.com/send-message
{
  "phone": "+2290196911346",
  "message": "Votre message ici"
}
```

### 2. `/send-file` (fichier + caption)

```json
POST https://psychobot-1si7.onrender.com/send-file
{
  "phone": "+2290196911346",
  "message": "Légende du fichier",
  "file_url": "https://tmpfiles.org/dl/ABC123/rapport.docx",
  "file_name": "rapport.docx",
  "mime_type": "application/vnd.openxmlformats-officedocument.wordprocessingml.document"
}
```

**Note**: Le fichier est d'abord uploadé sur `tmpfiles.org` (7 jours de rétention), puis l'URL est passée à PsychoBot qui télécharge et envoie via WhatsApp.

## 🛠️ Dépannage

### Erreur: `ModuleNotFoundError: No module named 'requests_toolbelt'`

```bash
python -m pip install requests-toolbelt
```

### Erreur: `HTTP 503 Bot not connected to WhatsApp`

- PsychoBot est déconnecté sur Render
- Allez sur https://psychobot-1si7.onrender.com et scannez le QR code

### Erreur: `Échec upload fichier: 413`

- Le fichier Word est trop lourd (> 50 MB)
- Compressez le document ou retirez les images haute résolution

### Fichier non reçu sur WhatsApp

- Vérifiez que PsychoBot a les permissions de télécharger des fichiers
- Essayez avec un fichier plus petit pour tester

## 📊 Cas d'Usage

### 1. Automatisation quotidienne

Créez un script `.bat` qui génère et envoie le rapport chaque matin:

```batch
@echo off
REM Générer le rapport TradingAgents
python scripts/generate_tradingagents_report.py --output "reports/daily_%date%.docx"

REM Envoyer sur WhatsApp
python Python/send_tradingagents_report.py --file "reports/daily_%date%.docx" --send-file

echo Rapport envoyé sur WhatsApp!
pause
```

### 2. Intégration avec webhook

Ajoutez à `ai_server.py` un endpoint qui envoie automatiquement après génération:

```python
@app.post("/tradingagents/generate-and-send")
async def generate_and_send_report(symbol: str):
    # Générer le rapport
    report_path = generate_tradingagents_report(symbol)
    
    # Envoyer via WhatsApp
    subprocess.run([
        "python", "Python/send_tradingagents_report.py",
        "--file", report_path,
        "--send-file"
    ])
    
    return {"success": True, "report": report_path}
```
