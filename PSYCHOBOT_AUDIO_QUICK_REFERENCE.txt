┌────────────────────────────────────────────────────────────────────────┐
│ 📊 PIPELINE DIAGRAM                                                    │
└────────────────────────────────────────────────────────────────────────┘

    🎙️ USER VOICE MESSAGE
         │
         ├─[1]─→ Download from WhatsApp (OGG Opus)
         │       📁 /tmp/voice_<id>.ogg
         │
         ├─[2]─→ Convert to WAV (FFmpeg)
         │       🔄 16kHz, mono, PCM
         │       📁 /tmp/voice_<id>.wav
         │
         ├─[3]─→ Transcribe (OpenAI Whisper)
         │       📝 French/English auto-detect
         │       💬 "Bonjour, quel est le statut de XAUUSD ?"
         │
         ├─[4]─→ AI Response (NVIDIA Llama 3.3 70B)
         │       🤖 Context-aware + Sidoine persona
         │       💬 "Yo ! XAUUSD est en tendance BULLISH..."
         │
         ├─[5]─→ Text-to-Speech (Google TTS)
         │       🎤 French voice, natural speed
         │       📁 /tmp/reply_<id>.mp3
         │
         ├─[6]─→ Convert to OGG (FFmpeg)
         │       🔄 OGG Opus (WhatsApp format)
         │       📁 /tmp/reply_<id>.ogg
         │
         ├─[7]─→ Send Voice Reply
         │       📤 WhatsApp voice message
         │       ✅ User receives audio response
         │
         └─[8]─→ Cleanup
                 🗑️ Delete all temp files
