# Instructions pour compiler et tester le scanner

## 1. Compilation dans MT5

1. Ouvrir MetaEditor (depuis MT5: Outils → MetaQuotes Language Editor)
2. Ouvrir le fichier: `SMC_Universal.mq5` depuis le Navigator
3. Appuyer sur F7 ou cliquer sur "Compiler"
4. Vérifier qu'il n'y a pas d'erreurs (les warnings sur POSITION_COMMISSION sont normaux)

## 2. Si erreur "file not found"

Le fichier a été copié automatiquement ici:
`C:\Users\USER\AppData\Roaming\MetaQuotes\Terminal\E6E3D0917DD641581E4779524EB3B1AA\MQL5\Experts\SMC_OpportunityScanner.mqh`

Si le problème persiste:
- Copier manuellement `SMC_OpportunityScanner.mqh` dans le dossier `MQL5/Experts/` ou `MQL5/Include/`
- Ou modifier la ligne d'include dans SMC_Universal.mq5 pour utiliser le chemin absolu

## 3. Test du scanner

1. Ouvrir plusieurs graphiques (ex: Boom 1000, Crash 1000, EURUSD)
2. Attacher SMC_Universal sur chaque graphique
3. Sur UN des graphiques, activer le scanner dans les paramètres:
   - EnableOpportunityScanner = true
   - ScannerSymbolsList = "Boom 1000 Index,Crash 1000 Index,EURUSD"
4. Le panneau scanner devrait s'afficher en haut du graphique

## 4. Vérification du fonctionnement

Le scanner devrait afficher:
- En-tête: "SCANNER OPPORTUNITÉS TEMPS RÉEL" (en or)
- Timestamp de mise à jour (coin droit)
- Liste des opportunités détectées (si disponibles)
- Message "Aucune opportunité détectée..." si aucun setup valide

## 5. Dépannage

### Panneau vide ou pas d'opportunités
- Vérifier que les robots tournent sur les autres graphiques
- Vérifier les noms des symboles (doivent être exacts)
- Attendre quelques minutes pour que les données se mettent à jour
- Vérifier les Global Variables (Outils → Global Variables dans MT5)

### Erreurs de compilation
- Vérifier que tous les fichiers .mqh sont présents
- Vérifier les chemins d'include
- Redémarrer MetaEditor

## 6. Configuration recommandée

```mql5
EnableOpportunityScanner = true
ScannerSymbolsList = "Boom 1000 Index,Crash 1000 Index,Volatility 75 Index"
ScannerRefreshSeconds = 2
ScannerPanelX = 10
ScannerPanelY = 30
ScannerPanelWidth = 500
ScannerShowPanel = true
```

## 7. Performance

- Le scanner utilise environ 1-2% de CPU supplémentaire
- Recommandé: maximum 10 symboles
- Si MT5 ralentit: augmenter ScannerRefreshSeconds à 3-5 secondes
