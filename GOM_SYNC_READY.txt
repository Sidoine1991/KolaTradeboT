WHAT IT DOES EVERY 10 MINUTES
================================================================================

1. LOAD
   └─ Fetches GOM verdicts from:
      Priority 1: MT5 live dashboard (/gom-kola-dashboard)
      Priority 2: AI server store (/gom-verdicts)
      Priority 3: Local JSON (data/gom_signal.json)

2. VALIDATE
   └─ Coherence ≥ 70%
   └─ Boom=BUY only, Crash=SELL only
   └─ Trading window UTC (7-17 XAUUSD, 8-22 BTC, etc.)
   └─ MTF coherence (4/6 timeframes aligned)
   └─ RSI extreme filter (no overbought/oversold)

3. POST
   └─ Each valid verdict → /gom-verdict on ai_server:8000
   └─ Enforces SL/TP safeguards for synthetics
   └─ Deduplicates by symbol

4. REPORT
   └─ Formats all active signals
   └─ Includes entry, SL, TP, coherence, timeframes

5. NOTIFY
   └─ Sends report via WhatsApp
   └─ Primary: ai_server /notify-whatsapp
   └─ Fallback: PsychoBot Render
