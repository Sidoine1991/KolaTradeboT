📋 DATA SOURCES
════════════════════════════════════════════════════════════════════════════

Dashboard Data:
   ✓ AI Signal: g_lastAIAction (real-time)
   ✓ Confidence: g_lastAIConfidence (0-100%)
   ✓ Trend: GetCurrentTrendDirection() (EMA-based)
   ✓ Price: SymbolInfoDouble(..., SYMBOL_BID)
   ✓ BUY Level: GetClosestBuyLevel() (support detection)
   ✓ SELL Level: GetClosestSellLevel() (resistance detection)
   ✓ ML Metrics: g_mlLastAccuracy, g_mlLastModelName

Projections:
   ✓ ATR: iATR(_Symbol, PERIOD_M1, 14)
   ✓ Levels: Current ± (ATR × multipliers)
   ✓ Time: 60 minutes into future
   ✓ Update: Every candle close


🚀 HOW TO USE
════════════════════════════════════════════════════════════════════════════

Reading the Dashboard:
   1. Check AI Signal color (GREEN = bias to BUY)
   2. Check Trend color (match with AI = strong signal)
   3. Current price position relative to entry levels
   4. ML accuracy tells you confidence in model

Setting Entries:
   • Watch for price approaching BUY level (GREEN line)
   • Watch for price approaching SELL level (RED line)
   • Use projection lines for TP/SL placement
   • BULL projection = TP target
   • BEAR projection = SL zone

Managing Trades:
   • Monitor real-time dashboard updates
   • Entry levels refresh every candle
   • Projections adjust with volatility (ATR)
   • Confidence zone shows expected range
   • Exit when projections invalidated
