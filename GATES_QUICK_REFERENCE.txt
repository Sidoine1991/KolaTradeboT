═══════════════════════════════════════════════════════════════════════════════
  THE 5 GATES
═══════════════════════════════════════════════════════════════════════════════

GATE 1: ML ACCURACY
  Blocks if: AI confidence < 75%
  Reason: Unreliable model = false signals
  Log: "🔴 GATE 1 BLOQUÉ: ML accuracy X% < 75%"

GATE 2: CONFIDENCE THRESHOLD
  Blocks if: Signal confidence < 80%
  Reason: Weak signal ≈ coin flip
  Log: "🔴 GATE 2 BLOQUÉ: Confidence X% < 80%"

GATE 3: MARKET STATE (ASIAN SESSION)
  Blocks if: Trading 22:00-08:00 London time
  Reason: Low liquidity + unpredictable = losses
  Log: "🔴 GATE 3 BLOQUÉ: Session Asiatique"

GATE 4: VOLATILITY
  Blocks if: ATR too low (< 0.0005 * Point)
  Reason: Choppy market = false breakouts
  Log: "🔴 GATE 4 BLOQUÉ: Volatilité insuffisante"

GATE 5: QUALITY SCORE
  Blocks if: Combined score < 75%
  Formula: (Confidence × 0.40) + (Score × 25) + (ML × 20) + (Vol × 15)
  Log: "🔴 GATE 5 BLOQUÉ: Signal quality score X%"
