📊 DONNÉES ÉCRITES À AWS RDS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TABLE: predictions
├─ symbol (ex: EURUSD)
├─ timeframe (ex: M1)
├─ action (buy/sell/hold)
├─ confidence (0.0-1.0)
├─ reason (technical analysis)
├─ metadata (JSON)
└─ created_at (ISO timestamp)

TABLE: model_metrics
├─ symbol
├─ accuracy (0.0-1.0)
├─ precision, recall
├─ models_loaded (count)
└─ timestamp

TABLE: trade_feedback
├─ symbol
├─ timeframe
├─ profit_usd
├─ is_win (boolean)
├─ side (buy/sell)
└─ executed_at
