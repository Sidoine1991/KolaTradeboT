WHAT'S DISPLAYED ON CHART
================================================================================

1. DIVERGENCE ZONES
───────────────────────────────────────────────────────────────────────────────
   • Green rectangle: BULLISH divergence zone (BUY signals)
   • Red rectangle: BEARISH divergence zone (SELL signals)
   • Transparent overlay (70% opacity) to see price action
   • Size: ±5 pips around signal price level

2. DIVERGENCE ARROWS
───────────────────────────────────────────────────────────────────────────────
   • Arrow code 233 (UP): Bullish divergence detected
   • Arrow code 234 (DOWN): Bearish divergence detected
   • Color: Green (buy) or Red (sell)
   • Width: 3 pixels (bold, easy to see)

3. CONFIDENCE LABELS
───────────────────────────────────────────────────────────────────────────────
   • Text: "BUY 87.5%" or "SELL 75.2%"
   • Shows direction + confidence percentage
   • Color: Matches arrow (green/red)
   • Size: 12pt font (prominent)

4. BLOCKED SIGNALS
───────────────────────────────────────────────────────────────────────────────
   • Text: "X (FILTERED)"
   • Gray color (muted)
   • Shows when signal blocked by gates
   • Helps identify what's being filtered out

5. SUPPORT & RESISTANCE LEVELS
───────────────────────────────────────────────────────────────────────────────
   • Green dashed line: Support (last 50 bars minimum)
   • Red dashed line: Resistance (last 50 bars maximum)
   • Width: 2 pixels
   • Updated every tick

6. MOVING AVERAGE (MA20)
───────────────────────────────────────────────────────────────────────────────
   • Blue dotted line: 20-period SMA
   • Shows trend direction
   • Width: 1 pixel (subtle)
   • Updated every tick

7. VOLATILITY BANDS (ATR)
───────────────────────────────────────────────────────────────────────────────
   • Orange dotted lines: Upper and lower ATR bands
   • Upper band: Current price + ATR(14)
   • Lower band: Current price - ATR(14)
   • Visualizes current volatility range
   • Width: 1 pixel each

HOW TO CUSTOMIZE VISUALS
================================================================================

1. Change Divergence Zone Colors:
   Line 470: zoneColor = (signal.direction == "BUY") ? clrGreen : clrRed;
   → Change clrGreen/clrRed to: clrBlue, clrYellow, clrCyan, etc.

2. Change Zone Transparency:
   Line 481: ObjectSetDouble(0, divergenceName, OBJPROP_TRANSPARENCY, 0.7);
   → Change 0.7 to 0.0 (opaque) ... 1.0 (invisible)

3. Change Line Styles:
   Line 523: STYLE_DASH → STYLE_SOLID, STYLE_DOT, STYLE_DASHDOT
   Line 531: STYLE_DASH → (same options)

4. Change Arrow Codes:
   Line 491: arrowCode = (signal.direction == "BUY") ? 233 : 234;
   → Use arrow codes 217-241 from MT5 documentation

5. Change Dashboard Position:
   Currently: Top-left (default Comment position)
   To move: Use ObjectCreate(..., OBJ_LABEL) instead of Comment()
