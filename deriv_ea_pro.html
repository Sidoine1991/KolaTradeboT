Guide de démarrage

1. Créez un compte Deriv.com et obtenez un token API (lecture + trading) dans les paramètres de votre compte.

2. Entrez votre token, choisissez Démo ou Réel, configurez mise et limites.

3. Testez d'abord sur le compte démo (sélectionné par défaut).

4. Le robot détecte automatiquement la catégorie d'actif et applique la stratégie optimale.

⚠️ Ce robot est fourni à titre éducatif. Le trading comporte un risque de perte. Ne tradez jamais avec plus que ce que vous pouvez vous permettre de perdre.
