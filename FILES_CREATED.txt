================================================================================
WORKFLOW
================================================================================

1. SCAN
   - GOM MT5 via /gom-verdicts
   - Filter: Good (±2) & Perfect (±3) only
   - Reject: WAIT, HOLD, others

2. VALIDATE BOOM/CRASH
   - SELL on Boom? NO
   - BUY on Crash? NO

3. ANALYZE EACH SIGNAL
   - Calculate Entry/SL/TP
   - Determine execution type (market/limit/stop)
   - Extract indicators (ATR, RSI, TF directions)

4. GENERATE REPORTS
   - Create Word report for each Good/Perfect
   - Send via WhatsApp

5. VALIDATE GATES (Top-3 only)
   - Gate 1: IA Status >= 70%
   - Gate 2: MTF alignment (H4/H1/M15)
   - Gate 3: MTF coherence >= 4/6 timeframes

6. PLACE ORDERS
   - Send to /pending-order endpoint
   - TradeManager executes order

7. NOTIFY
   - Send WhatsApp summary
   - Report: placed, errors, duration

================================================================================
QUALITY GATES
================================================================================

✓ Good/Perfect Filter: verdict_num ±2, ±3
✓ IA Status Gate: coherence_pct >= 70%
✓ MTF Gate: H4 BULL OR (H1 BULL & M15 BULL) for BUY
✓ MTF Coherence: >= 4/6 timeframes aligned
✓ Boom/Crash Rule: SELL forbidden on Boom, BUY forbidden on Crash

================================================================================
TROUBLESHOOTING
================================================================================

No signals found?
   Check: curl http://127.0.0.1:8000/gom-verdicts | grep -i good

Orders blocked (IA status)?
   Check: coherence_pct must be >= 70%

MTF gate failures?
   Check: H4/H1/M15 alignment, >= 4/6 timeframes

Reports not sent?
   Check: PsychoBot accessible, WhatsApp connected
