DÉTAILS DES ÉLÉMENTS VISUELS
═════════════════════════════════════════════════════════════════════════════════

1. BOUGIES PRÉDITES
   ┌─────────────────────────────────────────────┐
   │  ╔═══╗  ← Mèche haute (high)                │
   │  ║   ║                                       │
   │  ║ █ ║  ← Corps (open → close)              │
   │  ║   ║     Vert si close > open             │
   │  ╚═══╝     Rouge si close < open            │
   │    │                                         │
   │    └──── Mèche basse (low)                  │
   │                                              │
   │  85%  ← Label confiance (au-dessus)         │
   └─────────────────────────────────────────────┘

2. TRAJECTOIRE
   Point A ──────▲──────→ Point B
           │     │        (Segment jaune épaisseur 2)
           │     └─ Flèche direction (▲ ou ▼)
           └─ Ligne continue

3. ZONE DE PRÉDICTION
   ╔════════════════════════════════════╗
   ║                                    ║  ← Rectangle transparent
   ║  ⚡ ZONE PRÉDICTION ML (label)    ║     Couleur: gris foncé
   ║                                    ║     Alpha: 230 (très transparent)
   ║  [Bougies + Trajectoire ici]      ║
   ║                                    ║
   ╚════════════════════════════════════╝

4. POINTS CLÉS
   🎯 Target    = Cible finale (dernier point)
   🔄 Pivot     = Retournement de tendance
   📍 Support   = Niveau bas probable
   📍 Résistance= Niveau haut probable

5. LABELS DE CONFIANCE
   85%  72%  60%  51%  43%  36%  30%
   ▲    ▲    ▲    ▲    ▲    ▲    ▲
   │    │    │    │    │    │    └─ Bougie 7 (faible confiance)
   │    │    │    │    │    └────── Bougie 6
   │    │    │    │    └─────────── Bougie 5 (moyenne)
   │    │    │    └──────────────── Bougie 4
   │    │    └───────────────────── Bougie 3 (bonne)
   │    └────────────────────────── Bougie 2 (très bonne)
   └─────────────────────────────── Bougie 1 (excellente)

   Formule: Confiance = 85% × (0.85 ^ index)

═════════════════════════════════════════════════════════════════════════════════
COULEURS ET TRANSPARENCE
═════════════════════════════════════════════════════════════════════════════════

Zone de prédiction:
  Couleur: clrDarkSlateGray (gris foncé)
  Alpha:   230 (très transparent, on voit à travers)

Bougies haussières:
  Couleur: clrLimeGreen (vert citron)
  Alpha:   180 × (confidence/100) = plus transparent si moins confiant

Bougies baissières:
  Couleur: clrCrimson (rouge profond)
  Alpha:   180 × (confidence/100)

Trajectoire:
  Couleur: clrYellow (jaune vif)
  Épaisseur: 2 pixels
  Style: Ligne continue

Flèches:
  Couleur: clrYellow (même que trajectoire)
  Codes: 233 (▲) pour UP, 234 (▼) pour DOWN
  Taille: 2

Labels:
  Confiance: clrGray (gris) - Police 7pt
  Target/Pivot: clrYellow - Police 8pt
