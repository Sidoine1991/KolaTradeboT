================================================================================
  📊 COMMENT ÇA FONCTIONNE ?
================================================================================

1️⃣ PROTECTION PRÉVENTIVE (à l'ouverture)
   ▶ Calcul automatique du SL pour ne pas dépasser 3$
   ▶ Formule : max_sl = 3$ / (lot × contract_size × tick_value / point)

2️⃣ SURVEILLANCE CONTINUE (toutes les secondes)
   ▶ Script vérifie TOUTES les positions ouvertes
   ▶ Si perte >= 3$ → Fermeture automatique immédiate
   ▶ Logs détaillés de chaque fermeture

3️⃣ API REST (intégration externe)
   ▶ Endpoint : http://localhost:8000/robot/monitor/loss-limit
   ▶ Paramètre : max_loss (défaut 3.0)
   ▶ Retourne : rapport avec positions fermées

  🔧 CONFIGURATION

Option 2 : Variable d'environnement
  .env ou .env.supabase
  MT5_MAX_LOSS_PER_TRADE=3.0

Option 3 : Via l'API
  curl -X POST "http://localhost:8000/robot/monitor/loss-limit?max_loss=5.0"

Problème : "MT5 n'est pas connecté"
  ➜ Solution : Ouvrir MT5 et se connecter au compte
  ➜ Vérifier : Variables MT5_LOGIN, MT5_PASSWORD, MT5_SERVER dans .env

  ➜ Solution : pip install MetaTrader5

  ➜ Solution : Vérifier Python avec "python --version"
  ➜ Vérifier les dépendances : pip install -r requirements.txt
