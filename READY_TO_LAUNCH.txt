================================================================================
TROUBLESHOOTING
================================================================================

Q: "Could not connect to localhost:8000"
A: Server not running. Check START_LIVE.bat launched without errors.
   Try different port: python ai_server.py --port 8001

Q: "ModuleNotFoundError: ..."
A: Install missing dependency:
   pip install [module_name]

Q: "Port 8000 already in use"
A: Kill old process:
   taskkill /IM python.exe /F
   Then restart server with different port

Q: Robot doesn't show on chart
A: Check MT5 Journal tab for compilation errors
   Verify EnableAutoTrading = true in inputs

Q: No GOM levels showing
A: Increase BarsForTouchCount to 200
   Decrease TouchZoneATRPercent to 20
   Wait a few bars for scan to complete
